## üì¶ Installation & Setup

### üê≥ Quick Start with Docker (Recommended)

The easiest way to run SkillSync is with Docker. This will start all services (Frontend, Backend, MongoDB, and N8n) with a single command.

#### Prerequisites
- Docker Desktop installed and running
- At least 4GB RAM available for Docker

#### Quick Setup
1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd skillsync
   ```

2. **Configure environment**
   ```bash
   copy .env.docker .env
   ```
   Edit `.env` and add your OpenAI API key

3. **Start all services**
   ```powershell
   .\start-docker.ps1
   ```
   Or manually:
   ```bash
   docker-compose up -d
   ```

4. **Configure N8n workflow**
   - Open http://localhost:5678
   - Login: `admin` / `admin123`
   - Import `n8n-workflows/skillsync-roadmap-workflow.json`
   - Add OpenAI credentials
   - Activate the workflow

5. **Access the application**
   - Frontend: http://localhost:3000
   - Backend: http://localhost:5000
   - N8n: http://localhost:5678

**üìñ For detailed Docker setup, see [DOCKER-SETUP.md](DOCKER-SETUP.md)**

## ü§ñ AI + Automation Architecture

### Overview
SkillSync uses a powerful combination of AI and automation to generate personalized career roadmaps:

1. **User Input** ‚Üí User selects career field and skill level
2. **n8n Trigger** ‚Üí Webhook receives request from frontend
3. **AI Generation** ‚Üí OpenAI GPT generates structured roadmap
4. **Data Processing** ‚Üí n8n transforms AI output into JSON + documentation
5. **Storage** ‚Üí Data saved to MongoDB via Appwrite API
6. **Visualization** ‚Üí Frontend renders interactive charts and documentation

### n8n Workflow Components

#### 1. Webhook Trigger
- **Endpoint**: `https://n8n.yourdomain.com/webhook/roadmap`
- **Method**: POST
- **Payload**: `{ field: "Web Development", level: "Beginner" }`

#### 2. AI Content Generation
- **Node**: OpenAI GPT-4
- **Prompt Template**:
  ```
  Generate a career roadmap for {{ $json.field }} at {{ $json.level }} level.
  Output in JSON with two parts:
  1. roadmap_json: structured roadmap for chart rendering
  2. roadmap_doc: documentation text in markdown
  ```

#### 3. Data Transformation
- **Function Node**: Parses AI output and formats data
- **Output**: Separate JSON for charts and markdown for documentation

#### 4. Database Integration
- **HTTP Request Node**: Calls backend API endpoints
- **Operations**: Save roadmap data and documentation

### Roadmap Data Structure

```json
{
  "roadmap_json": {
    "title": "Web Development Career Path",
    "level": "Beginner",
    "duration": "12-18 months",
    "phases": [
      {
        "id": "phase-1",
        "title": "Foundation",
        "duration": "3 months",
        "steps": [
          {
            "id": "step-1",
            "title": "HTML & CSS Basics",
            "description": "Learn markup and styling fundamentals",
            "resources": ["MDN Docs", "freeCodeCamp"],
            "estimatedHours": 40,
            "prerequisites": [],
            "skills": ["HTML", "CSS", "Responsive Design"]
          }
        ]
      }
    ],
    "connections": [
      { "from": "step-1", "to": "step-2" }
    ]
  },
  "roadmap_doc": "# Web Development Career Roadmap\n\n..."
}
```

## üîß API Endpoints

### Roadmaps (AI-Generated)
- `POST /api/roadmaps/generate` - Generate new AI roadmap
- `GET /api/roadmaps` - Get all user roadmaps
- `GET /api/roadmaps/:id` - Get roadmap by ID
- `PUT /api/roadmaps/:id` - Update roadmap progress

- `POST /api/roadmaps/:id/rate` - Rate roadmap
- `GET /api/roadmaps/fields` - Get available career fields

- `POST /webhook/roadmap` - Trigger AI roadmap generation

## ü§ñ n8n Automation Setup

### Prerequisites
- n8n instance (self-hosted or cloud)
- OpenAI API key
- Backend API accessible from n8n

### Setting Up n8n Workflows

1. **Import Workflow Files**
   ```bash
   # Copy workflow files to your n8n instance
   cp n8n-workflows/*.json /path/to/n8n/workflows/
   ```

2. **Configure Environment Variables**
   - `OPENAI_API_KEY` - Your OpenAI API key
   - `BACKEND_API_URL` - Your backend API base URL
   - `WEBHOOK_SECRET` - Secret for webhook authentication

3. **Activate Workflows**
   - Import `roadmap-generator.json` into n8n
   - Configure OpenAI node with your API key
   - Set webhook URL in your frontend
   - Test the workflow with sample data

### Workflow Endpoints
- **Roadmap Generation**: `https://your-n8n.com/webhook/roadmap`
- **Content Updates**: `https://your-n8n.com/webhook/content-update`

### Example Frontend Integration
```javascript
// Trigger AI roadmap generation
const generateRoadmap = async (field, level) => {
  const response = await fetch('https://your-n8n.com/webhook/roadmap', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ field, level, userId: user.id })
  });
  return response.json();
};
```
